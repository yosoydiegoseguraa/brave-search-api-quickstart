# Brave Search API: Python Quickstart 🛠️

This repository serves as a high-utility guide for developers to integrate Brave’s independent search index into their AI pipelines in under 60 seconds.

## The Problem
Most AI search wrappers rely on Big Tech indices that harvest user data and offer limited transparency. 

## The Brave Solution
* **Zero-Data-Retention (ZDR):** Privacy is at the protocol level, not just a promise.
* **Independent Index:** Not a wrapper. We crawl, index, and serve our own data.
* **RAG-Ready:** JSON responses optimized for LLM context window injection.

## Getting Started
1. `pip install requests`
2. Add your API key to `brave_search_quickstart.py`
3. Run `python brave_search_quickstart.py`
